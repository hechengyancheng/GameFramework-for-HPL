# 迷雾森林传说 - 游戏内容丰富更新日志

## 更新日期
2024年

## 主要更新内容

### 1. 新增职业系统
- **战士**: 高HP(150)，擅长近战，初始装备生锈的剑和皮甲
- **法师**: 高MP(100)，擅长魔法，初始装备魔法杖和法师袍
- **游侠**: 平衡型，擅长远程，初始装备精灵弓和皮甲

### 2. 新增场景（4个）
- **洞穴深处**: 更危险的洞穴区域，有洞穴巨魔和远古宝石
- **古老遗迹**: 神秘的古代遗迹，需要古老钥匙才能进入深处
- **黑暗沼泽**: 危险的沼泽地，有沼泽怪和发光蘑菇
- **商人营地**: 可以购买装备和出售材料
- **巨龙巢穴**: 最终Boss战场景，挑战红龙成为屠龙者

### 3. 新增物品（18种）
**武器**:
- 铁剑（攻击力12）
- 魔法杖（攻击8，魔法15）
- 精灵弓（攻击10）

**防具**:
- 皮甲（防御5）
- 铁甲（防御12）
- 法师袍（防御3，魔法10）

**消耗品**:
- 魔法药水（恢复30MP）
- 解毒剂（解除中毒）
- 力量药剂（临时增加攻击力）

**任务物品**:
- 古老钥匙（打开遗迹石门）
- 神秘卷轴（记载古代魔法）
- 龙鳞（巨龙掉落，价值500）

**稀有材料**:
- 魔法水晶（价值80）
- 远古宝石（价值300）
- 发光蘑菇（价值15）
- 月光花（价值40）

### 4. 游戏机制增强
- **商店系统**: 可以购买生命药水(20金)、魔法药水(25金)、铁剑(120金)、皮甲(80金)
- **随机事件**: 森林搜索、洞穴探索、沼泽冒险都有多种随机结果
- **条件判断**: 遗迹石门需要古老钥匙才能打开
- **Boss战系统**: 巨龙巢穴有谈判、挑战、偷取三种选择
- **任务提示**: 村庄公告板提供任务信息

### 5. 场景丰富
**迷雾森林**:
- 新增通往遗迹和沼泽的路径
- 搜索功能增强：可能发现草药、包裹、陷阱或什么都没有
- 新增月光花采集（夜间）

**幽暗洞穴**:
- 新增洞穴深处入口
- 战斗系统增强：蝙蝠群、洞穴蜘蛛、巨型蝙蝠
- 新增火把搜索功能

**宁静村庄**:
- 新增商人营地入口
- 新增铁匠铺访问
- 新增公告板查看任务
- 新增休息功能（恢复全部HP/MP，花费10金）
- 探索功能增强

### 6. 游戏流程
1. 选择职业（战士/法师/游侠）
2. 探索迷雾森林，收集资源
3. 进入洞穴或前往村庄
4. 在商人营地购买装备
5. 探索遗迹（需要钥匙）或沼泽
6. 挑战巨龙巢穴，成为屠龙者！

## 游戏特色
- 三种职业，不同玩法体验
- 7个场景，丰富的探索内容
- 20+种物品，完整的装备系统
- 随机事件，每次游戏都不同
- 最终Boss战，具有挑战性

- 使用HPL声明式语法
- 模块化设计，易于扩展
- 随机数系统增加游戏性
